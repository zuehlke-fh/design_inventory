# Learnings

## Description
We feel that the entire course was a huge learning experience. Coming in, we didn’t have much knowledge about electronics, 3D modeling, or similar fields. But despite that, it was incredibly interesting to take part in this journey.

One of the biggest lessons we learned: No matter how fast you think you’ll be, it will always take twice as long. We often met up, confidently saying, “This will be done in no time.” Oh, sweet summer child... it was not. We spent way to much time trying to figure out why something that had worked perfectly before suddenly didn’t—only to find out that a single cable wasn’t properly soldered.

Another key takeaway: Debug everything—even if it’s not code. Installing a status LED turned out to be a game-changer. If the LED doesn’t light up, you instantly know there’s no electricity flowing, saving hours of unnecessary troubleshooting.

In the end, this project was as much about learning how to problem-solve as it was about building something cool.
