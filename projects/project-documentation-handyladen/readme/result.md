# Results

## Description
The result of this project is a fully functional prototype that meets all the expectations we set. The power generated from riding the bike is more than sufficient to reliably charge the power bank. The power output from the power bank powers all the functions of the prototype: the phone charges at 5 volts, and the light is bright enough to improve safety during bike rides. A long press on the button allows the user to switch between a steady light source or a blinking on-off mode.

All materials used in this project were sourced from old components belonging to friends and family, which would have otherwise been discarded. This gave those materials a second life.

The prototype is now ready for further development as a bike phone charger, addressing a clear demand for such a product.
