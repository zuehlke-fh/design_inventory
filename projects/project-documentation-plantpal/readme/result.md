# Result: [PlantPal]

## Description
In this section we describe the results of the project and how the prototype worked.

## Content

How the system works: The prototype works as planned. The moisture sensor measures the soil moisture level and activates the irrigation when it falls below the threshold. The relay ensures that power is always available, either from the generator or the power supply unit.

Power supply: Switching between the power sources works well. The battery can be charged with the generator and the system automatically switches to the mains adapter when the battery is empty.

Test results: The system proved to be reliable in the tests and ensures that the plant always gets water at the right time. The adjustments to the sensors and the power supply have improved stability.
