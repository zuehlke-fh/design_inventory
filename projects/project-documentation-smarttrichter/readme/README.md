## Basic Topic Description
The **Smart Trichter** is a party device that measures drinking speed and displays the results on a screen. Users can register using their student ID to save and compare their results. The device is battery-powered, requires no external power source, and allows for simple data input via a keyboard. It is designed for **fun, non-alcoholic drinking competitions**.

For hygiene reasons, the device features an **interchangeable mouthpiece**, ensuring a safe and sanitary experience for multiple users.

## Idea and Inspiration
This project combines interactive physical prototyping with a unique social gaming experience, inspired by the need for engaging group activities. The concept aims to enhance competition and fun in a safe and inclusive manner.
