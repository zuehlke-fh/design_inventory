# Construction: [LockQuest]

## Description
The construction is split in two parts:
- Electronic part
- Woodcrafting part

## Content
### Electronic part
The core of the circuit is an esp32 and it connects to all other electronic components.\
Pin A3 and A4 are connected to hand trainers with pull-down resistors.\
Pin A5 is connected to the sound module (buzzer).\
Pin 8 is connected to the servo.\
SCL and SDA are connected to the gyro sensor.
