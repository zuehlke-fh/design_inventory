# Learnings: [LockQuest]

As part of our “LockQuest” project, we have gained valuable insights in various areas:

Sensor integration:

Initially, we faced challenges in implementing the gyro sensor. After intensive troubleshooting and adjustments, we were able to successfully integrate the sensor into the ESP32, which means it now works reliably.

Sound module and servo:

An unexpected problem arose when the sound module impaired the function of the servo. Through targeted analysis and optimization of the circuit logic, we were able to resolve this problem and ensure that both components work together smoothly.

Hardware consolidation:

Thanks to Schratti's commitment, a customized chest was developed to efficiently house all electronic components, improving hardware handling and protection.

Handle tools:

Instead of the originally planned electrodes, we opted for hand trainers that are connected with cables, which act simmilar to buttons when pressed together. This adaptation proved to be more practical and user-friendly.

Functionality:

The cooperation between the gyro sensor and the grip tools is now reliable. In addition, the sound module and the servo are synchronized, resulting in an improved user experience.
