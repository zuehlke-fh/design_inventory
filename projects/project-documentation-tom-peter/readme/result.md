# Result

The final result is a fun trumpet-inspired MIDI controller that lets users play virtual instruments with more expression. It sends MIDI note, velocity, and pitch data to DAWs or synthesizers and feels responsive and intuitive.
## Key Features
- Natural Playability: Valve buttons and breath control made it intuitive for trumpet players to use.
- Breath-Controlled Expression: The breath sensor added dynamic control over volume and expression, mimicking the feel of a real trumpet.
- Pitch Modulation: A rotary sensor allowed for pitch bends and smooth glissandos.
- Seamless DAW Integration: Plug-and-play USB MIDI worked with DAWs like Ableton and Logic without extra setup.
- Creative Sound Control: Mapped breath and pitch input allowed for both realistic brass sounds and experimental synth effects.
## Areas for Improvement
- Advanced Pitch Control: Future versions could explore more refined pitch modulation techniques, such as using capacitive touch sensors.
- Wireless MIDI: Adding Bluetooth or other wireless options would make the controller more versatile.
- Frame Design: A more polished, durable frame could improve aesthetics and robustness.
