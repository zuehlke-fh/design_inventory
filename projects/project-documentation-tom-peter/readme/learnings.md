# Learnings
## Working with Teensy 4.0:
Connecting the Controller to our PC's was challenging:
At first the Controller wasn't recognized by our Laptops because we were using the wrong USB cable.
After that our code was giving out errors mentioning the Library "Control-Surface". With that, we couldn't compile code on our Teensy.
## Circuit Board Assembly
No one in our team has ever soldered before, we learned the basics and, while really ugly at first, got the hang of soldering after a while.
The professor showed us how to undo a wrong solder. 

- A LED diode was destroyed due to a missing resistor.
- Grounding and voltage connections were initially forgotten during testing.

## Other Software
While integrating Ableton Live Lite, we had problems with sending data back to the Teensy because it caused the Teensy to crash.
We fixed it by adding an empty read method.
