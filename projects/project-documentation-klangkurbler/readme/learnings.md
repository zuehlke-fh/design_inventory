# Learnings

Through building Klangkurbel, we gained hands-on experience on integrating hardware and software using the ESP32 Feather and its Music Maker wing.

We learned how to set up a web interface on the ESP and also how to manage real-time sensor input. By soldering the electrical board, buidling a voltage regulator and a voltage devider, experience in electronics could also be gained.

Construction should have been planned out better, since we had to cut a few features because of bad time management.

Altogether, we are very pleased with the result.
