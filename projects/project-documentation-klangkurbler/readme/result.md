# Result

This project involves an ESP32-based audio player powered by a crank generator, which also charges the ESP32's battery. The system integrates an Adafruit VS1053 MP3 codec for audio playback and features an SD card for storing music files.

Key functionalities include:

- Web Interface: Users can upload and delete songs via a web server hosted on the ESP32.
- Playback Control: The power generated by the crank is used to adjust playback behavior.
- Automatic Track Selection: The device detects and plays audio files sequentially from the SD card.
- Standalone Power: The crank generator ensures continuous operation without external power sources.

This setup creates an off-grid, manually powered music player with remote file management capabilities.
